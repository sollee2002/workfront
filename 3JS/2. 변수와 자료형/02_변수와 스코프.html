기본문법

1) 전역변수와 지역변수

  • 전역변수와 지역변수

    전역변수 : 함수 외부에서 선언 (어디서든 사용 가능)

    지역변수 : 함수 내부에서 선언 (함수 내부에서만 사용 가능)

2) 변수와 스코프

  • 함수레벨 스코프(var)

    함수 외부에서 함수 내부에서 선언된 변수에 접근 불가

  • 블럭레벨 스코프(let, const)

    블럭 외부에서 블럭 내부에 선언된 변수에 접근 불가
    *주의 : 예약어를 붙이지 않은 경우 오류가 발생하지 않으며 전역 변수로 생성됨

var 예약어의 문제점

  • -함수의 코드 블록만을 스코프로 인정한다

    -함수 외부에서 생성한 변수는
